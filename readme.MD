Proyek ini adalah website portofolio pribadi responsif yang dibangun menggunakan HTML5, CSS3 modern, dan Bootstrap 5 untuk menampilkan identitas profesional, keahlian pemrograman, serta dokumentasi proyek mahasiswa Teknik Informatika Universitas Ciputra Makassar. Dengan integrasi animasi reveal, galeri foto yang dinamis, dan desain antarmuka yang bersih, proyek ini berfungsi sebagai etalase digital untuk menampilkan kapabilitas teknis dalam pengembangan web serta manajemen aset media secara efisien.
